Noundry.AIG

Unified AI Gateway for .NET - Access OpenAI, Anthropic, Google Gemini, and more through a single, consistent API.

dotnet add package Noundry.AIG.Client

Features

  • Multi-provider support (OpenAI, Anthropic, Google)
  • Automatic failover and provider ordering
  • Streaming responses & PromptBuilder API
  • ChainPromptBuilder for multi-step AI workflows
  • Thread-safe, production-ready client
  • Built-in analytics and logging with Tuxedo
